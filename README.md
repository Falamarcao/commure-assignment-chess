# Commure Assiggnment Chess

![Commure Assiggnment Chess](./docs/images/screenshot.png)

## Install dependencies with poetry

```sh
    poetry install
```

## Run tests

```sh
    poetry run python -m unittest discover
```

## Run Project

```sh
    poetry run ./commure_assignment_chess/main.py
```
